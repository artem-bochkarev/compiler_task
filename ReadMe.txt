Проект создан в Eclipse, но и другие Java IDE должны с ним работать.

Список классов:

ParserHandMade - рукописный парсер (ранее была автогенерируемая версия Parser.java)
LexerCreator   - создает класс Lexer из файла грамматики
Tokencreator   - создает класс Token
TreeCreator    - создает класс Tree (дерево разбора)
Result         - обертка над деревом разбора с добавлением информации об ошибках
Tester         - запускает обработку файла simple.c. Lexer, Tree, Token генерируются заново по файлу c.g
                 Вносить изменения в этот файл не рекомендуется, т.к парсер теперь не генерируется автоматически,
                 поэтому изменения в разделе грамматики никак не обрабатываются, а вносятся туда лишь для наглядности,
                 а изменения в разделе Лексера могут привести к ошибкам выполнения.

Остальные классы остались от старых версий программы и теперь не используются.
Реализовано восстановление после ошибок.
